## Description

Add keyboard shortcuts for common actions to improve power-user efficiency. Implement global search via Cmd/Ctrl+K, Escape to close modals, arrow key navigation for the sidebar, and a help overlay accessible via the ? key.

---

## Acceptance Criteria

- [ ] `Cmd/Ctrl + K` navigates to `/search` and focuses the search input
- [ ] `Escape` closes any open modal (help overlay, mobile sidebar)
- [ ] Arrow keys navigate sidebar items when sidebar is focused
- [ ] A keyboard shortcut help overlay is accessible via `?` key
- [ ] Shortcuts do not fire when user is typing in an input/textarea/contenteditable
- [ ] Shortcuts do not conflict with browser defaults
- [ ] `npm run build` and `npx vitest run` pass with no regressions

---

## Technical Context

### Current State
- No keyboard shortcuts exist in the app
- Global search page exists at `src/app/search/page.tsx` with an `autoFocus` input
- Sidebar (`src/components/Sidebar.tsx`) has a `NAV_ITEMS` array with hrefs
- Layout at `src/app/layout.tsx` wraps `StoreProvider` > `Sidebar` + `<main>`
- No hooks directory structure (only `src/hooks/useStore.ts`)

### Implementation Plan

1. **`src/hooks/useKeyboardShortcuts.ts`** -- Custom hook for global keyboard shortcuts
   - Listen for `keydown` events on `document`
   - Guard against input/textarea/contenteditable focus
   - `Cmd/Ctrl+K`: navigate to `/search`, prevent default browser search bar
   - `Escape`: close help overlay, close mobile sidebar
   - `?`: toggle help overlay
   - Arrow up/down: navigate sidebar items when sidebar is focused

2. **`src/components/KeyboardShortcutHelp.tsx`** -- Help overlay component
   - Modal overlay showing all available shortcuts
   - Grouped by category (Navigation, General)
   - Styled with project sage/stone palette
   - Close via Escape or clicking outside

3. **`src/components/KeyboardShortcutProvider.tsx`** -- Client component wiring
   - Wraps the app in `layout.tsx`
   - Provides context for help overlay visibility
   - Integrates the `useKeyboardShortcuts` hook

4. **`src/app/layout.tsx`** -- Wire in the provider

5. **`src/hooks/__tests__/useKeyboardShortcuts.test.ts`** -- Tests for shortcut logic
