## Description

Export the baby items list as a CSV file for sharing with partners, family, or for use in spreadsheets. The export respects the current view mode (All Items vs My Tracked Items) and any active filters, so users can export exactly what they see on screen. The CSV uses RFC 4180 escaping to ensure compatibility with Excel, Google Sheets, and other spreadsheet tools.

---

## Acceptance Criteria

- [ ] "Export CSV" button on the Items page triggers a CSV file download
- [ ] CSV includes columns: Name, Category, Priority, Timing, Status, Estimated Cost, Actual Cost, Notes, Link
- [ ] Filename includes a timestamp (e.g., `baby-items-2026-04-21.csv`)
- [ ] Both "All Items" and "My Tracked Items" views export the currently visible (filtered) items
- [ ] CSV uses proper RFC 4180 escaping for fields containing commas, quotes, or newlines
- [ ] Unit tests verify CSV generation, escaping edge cases, and timestamped filename
- [ ] `npm run build` and `npx vitest run` pass with no regressions

## Technical Context

### Current State
- `src/app/items/page.tsx` -- Items page with `mergedItems`, `filtered`, view mode toggle, and filter controls
- `src/lib/csvImporter.ts` -- Existing CSV parser (import direction); the exporter mirrors its column conventions
- `src/types/index.ts` -- `BabyItem` interface with all relevant fields

### Implementation Plan

1. **`src/lib/csvExporter.ts`** -- New module for CSV export logic
   - `escapeCsvField(value: string): string` -- RFC 4180 field escaping (wrap in quotes if field contains comma, quote, or newline; double-escape internal quotes)
   - `exportItemsToCsv(items: ExportableItem[]): string` -- generates CSV string with header row and data rows
   - `downloadCsv(csvContent: string, filename: string): void` -- triggers browser file download via Blob + anchor click
   - `generateExportFilename(): string` -- returns `baby-items-YYYY-MM-DD.csv` with current date
   - Pure logic separated from UI for testability

2. **`src/app/items/page.tsx`** -- Modifications
   - Import `Download` icon from lucide-react
   - Import export utilities from `csvExporter.ts`
   - Add "Export" button in the header button group
   - On click: map `filtered` items to exportable rows and trigger download

3. **Tests** -- `src/lib/__tests__/csvExporter.test.ts`
   - Basic CSV generation with header and data rows
   - Fields containing commas are properly quoted
   - Fields containing double quotes are escaped with doubled quotes
   - Fields containing newlines are properly quoted
   - Empty/undefined fields render as empty strings
   - Filename includes current date
   - Empty items array produces header-only CSV
