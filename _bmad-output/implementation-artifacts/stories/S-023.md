# S-023: Printable Hospital Bag Checklist

**Epic:** E-005 (Import, Export, and Print Enhancements)
**Sprint:** 4
**Story Points:** 2
**Status:** In Progress
**Priority:** P2

## Description

Generate a print-friendly view of the hospital bag checklist that can be printed directly from the browser. Add a Print Checklist button that triggers the browser print dialog, and use `@media print` CSS rules to create a clean layout with categories and checkboxes.

## Acceptance Criteria

- [ ] "Print Checklist" button on the Hospital Bag page opens browser print dialog
- [ ] Print layout groups items by category with checkboxes
- [ ] Packed items show as checked checkboxes, unpacked as empty checkboxes
- [ ] Print layout hides navigation, buttons, and other non-essential UI
- [ ] Uses `@media print` CSS rules
- [ ] Item quantity and notes are visible in print layout
- [ ] Page title "Hospital Bag Checklist" appears at top of printed page

## Tasks

1. Add a "Print Checklist" button to the Hospital Bag page header (next to Pack All and Add Item)
2. Add `print-hide` class to non-essential UI elements (buttons, progress bar, modals)
3. Add print-specific CSS in globals.css for the hospital bag checklist layout
4. Render print-only checklist view with Unicode checkbox characters
5. Ensure items show quantity and notes in print view

## Technical Notes

- The existing `@media print` block in globals.css already hides `aside` and `.print-hide` elements
- Use `window.print()` to trigger the browser print dialog
- Use Unicode characters for checkboxes: ☑ (checked) and ☐ (unchecked)
- Add a `print-only` class for elements that should only appear when printing
- The Printer icon from lucide-react is appropriate for the button
